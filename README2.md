# 📈 Stock Market Dashboard in Python

An interactive stock market dashboard built with **Streamlit**, **Plotly**, and **yFinance**.  
This project demonstrates real-time data visualization, technical indicators, and trading signals in a professional dashboard layout.

---

## 🚀 Project Overview
The dashboard allows users to:
- Enter any stock ticker (e.g., AAPL, TSLA).
- Select a custom date range.
- View candlestick charts with Bollinger Bands.
- Track RSI, MACD (with histogram), and trading volume.
- Get automatic BUY/SELL/HOLD signals with clear reasoning.
- See market status (🟢 Open / 🔴 Closed) in real time.
- Enable live auto-refresh for continuous updates.

---

## ✨ Features
- 📈 **Price Card**: Current price, today’s high/low, and volume.
- 📊 **Statistics**: Average/Highest/Lowest close, average/highest volume, latest RSI & MACD.
- 💹 **Trading Signal**: Rich explanations (RSI levels, MACD crossover, Bollinger Band context).
- 📉 **Charts**:
  - Candlestick with Bollinger Bands
  - Volume bar chart
  - RSI chart with 70/30 lines
  - MACD chart with Signal line + Histogram
- 🟢🔴 **Market Status**: Sidebar shows if market is open or closed.
- 🔄 **Live Mode**: Auto-refresh at user-defined intervals.
